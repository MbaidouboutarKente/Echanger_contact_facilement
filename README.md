# Site Web ContactSphere

Site web officiel pour l'application mobile ContactSphere - Votre carnet de contacts intelligent.

## Description

Ce site web présente l'application ContactSphere qui permet de gérer, partager et transférer vos contacts de manière intelligente grâce aux QR codes, à la détection SIM et aux instructions USSD.

## Fonctionnalités du site

- Design responsive moderne
- Présentation des fonctionnalités de l'application
- Section téléchargement avec boutons vers les stores
- Formulaire de contact interactif
- Animations au scroll
- Navigation mobile optimisée
